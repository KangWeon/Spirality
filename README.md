## Spirality - 螺旋绘画
> 启发于微软平台的 [Spirality](https://www.microsoft.com/en-us/store/p/spirality/9pgtlzjj1rs6)。由于它的魔性，用来模仿探索画笔记录、橡皮擦、历史播放等等功能。

## 功能
- [x] 颜色选取
- [x] 画笔阵列
- [x] 画笔类型切换
	- [x] 钢笔 
	- [x] 水桶 
	- [ ] 对称 
- [x] 图层清除
- [x] 画笔记录撤销、恢复
- [x] 保存
	- [x] 保存为图片
	- [ ] 恢复上次未完成记录
	- [ ] 保存历史作画记录
- [x] 播放绘画过程
	- [ ]	停止播放
